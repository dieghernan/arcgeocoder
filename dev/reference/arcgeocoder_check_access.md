# Check access to ArcGIS REST

Check if **R** has access to resources at ArcGIS REST API
<https://developers.arcgis.com/rest/geocode/api-reference/overview-world-geocoding-service.htm>.

## Usage

``` r
arcgeocoder_check_access()
```

## Value

a logical.

## Examples

``` r
# \donttest{
arcgeocoder_check_access()
#> [1] TRUE
# }
```
