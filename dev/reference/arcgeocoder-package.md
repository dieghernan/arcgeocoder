# arcgeocoder: Geocoding with the 'ArcGIS' REST API Service

Lite interface for finding locations of addresses or businesses around
the world using the 'ArcGIS' REST API service
<https://developers.arcgis.com/rest/geocode/api-reference/overview-world-geocoding-service.htm>.
Address text can be converted to location candidates and a location can
be converted into an address. No API key required.
